Biopsy
  • If polyp removed via "cold biopsy forceps or cold snare" then NO biopsy, unless phrased as "cold biopsy forceps to perform biopsy", NO biopsy;
  • if cold forceps used to remove polyp, procedure is polypectomy, NO biopsy;
  • If a biopsy forcep (hot/cold) is used just to remove a polyp (polypectomy) code NO;
  • If they take a biopsy of large polyp without removing polyp code NO;
  • Base this variable on colonoscopy report only (e.g. if colonoscopy does not report biopsy but there is associated path report then code as NO biospy);
  • Biopsy of ulcers = YES
  • "Await biopsy report" - anything in the plan does not imply any label. “Plan” ,”impressions”, or “follow-up” should not be read for making a judgement.
Appendiceal orifice
  • they see orifice = YES ;
  • enter ileum = YES;
  • mention "cecal landmarks", "typical landmarks", "landmarks of cecum", "characteristic anatomy" = YES;
  • ileocolonic anastamoses (cecum missing), cecal pouch = YES;
  • if see "cecal tip" = YES ;
  • if they see appendix = YES;
  • Mark NO if it says "Appendiceal orifice was not reached", or if none of the above match.
  • A mention of cecum alone is insufficient here.
